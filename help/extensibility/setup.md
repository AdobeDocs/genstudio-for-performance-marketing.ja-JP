# 拡張可能なアプリの構築を始める

GenStudio for Performance Marketing _アドオン_&#x200B;は、組織のコンテンツの作成と検証をサポートするプロダクト機能を拡張するために作成するAdobe App Builder アプリケーションです。 GenStudioのエコシステムでは、「App Builder apps」、「extensions」、「extensible apps」という用語はすべて「Add-ons」を指します。

GenStudio for Performance Marketingアドオンフレームワークは、GenStudio for Performance Marketingのコンテンツ制作ワークフローや検証に、独自の請求処理コンプライアンスのプロトコルを組み込むためのツールを提供します。 このツールセットを使えば、開発者はカスタムアプリやアドオンを作成できます。

## 概要

このガイドでは、GenStudio for Performance Marketingの主要なAI駆動型コンテンツ制作機能をカスタマイズおよび強化するためのアドオンを作成するプロセスについて説明します。

GenStudio for Performance Marketing向け拡張可能アプリの開発者は、標準のAdobe CX エンタープライズアプリ開発ツールセットフレームワークを使用して、カスタムアプリまたはアドオンを開発およびデプロイします。 [Adobe App Builder](https://developer.adobe.com/app-builder/)は、この開発ツールセットの中心です。

アドオンの作成には、次の上位タスクが含まれます。

* [App Builder プロジェクトの作成](create-project.md)
* [App Builder アプリケーションの作成と開発](create-app.md)
* [アプリをデプロイ](deploy-app.md)
* [アプリの配布](distribute-app.md)

## 環境の設定

開発者は、Adobe App Builderを使用して、GenStudio for Performance Marketing用のカスタムアドオンを構築します。 Adobe Creative エクスペリエンス製品用のカスタムアプリを初めて開発する場合は、Adobe App Builder フレームワークを熟知してください。 チュートリアルとディスカッションについては、[App Builder ドキュメント &#x200B;](https://developer.adobe.com/app-builder/docs/overview/?lang=ja)を参照してください。

### 開発のためのツール

アドオン開発には以下のツールとリソースが含まれます：

[Adobe App Builder](https://developer.adobe.com/app-builder/)

[Adobe GenStudio UI拡張性SDK](https://github.com/adobe/genstudio-uix-sdk)

[ADOBE UIX SDK](https://github.com/adobe/uix-sdk)

[React Spectrum](https://react-spectrum.adobe.com/react-spectrum/getting-started.html)

### 環境

Adobe App Builderは、GenStudio for Performance MarketingがプロビジョニングされたIMS組織と同じ組織で、デフォルトで有効になっています。

環境を設定したら、[App Builder プロジェクトを作成して](create-project.md) アプリを作成できます。
