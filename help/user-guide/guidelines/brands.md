# GenStudio for Performance Marketing [!DNL Brands]

企業は、組織、製品、サービス、コンセプトを他の企業と区別します。 ブランドの要素には、ロゴなどの客観的なものもあれば、トーンなどの主観的なものもあります。

GenStudio for Performance Marketingは、[!DNL Brand]の豊富なブランド情報を使用して、[!DNL Product]および[!DNL Persona]情報と組み合わせ、包括的なブランドアイデンティティを構築します。 このブランドアイデンティティは、Adobeの生成AI テクノロジーを活用して、ブランドに即したコンテンツの制作に役立てるために使用されます。

## ブランド管理

GenStudio for Performance Marketingにブランドガイドラインを追加した後、[管理](/help/user-guide/guidelines/add-guidelines.md#manage-brands)および[公開](/help/user-guide/guidelines/add-guidelines.md#publish-brand)できます。 公開すると、アセット生成でチームが使用できるようになります。

[!DNL Brands]は、_未公開_&#x200B;または&#x200B;_公開済み_&#x200B;の状態にすることができます。 新規[!DNL Brands]は&#x200B;_未公開_&#x200B;として開始されます。

チームがアセット生成で利用できるようにするには、[a [!DNL Brand]](/help/user-guide/guidelines/add-guidelines.md#publish-brand)を公開する必要があります。

公開したブランドは編集（変更はすぐに利用できます）、複製、または非公開にできます。 コンテンツの作成と検証に使用できるのは、公開された[!DNL brands]のみです。 「[&#x200B; ガイドラインを追加](/help/user-guide/guidelines/add-guidelines.md#manage-brands)」を参照してください。

削除された[!DNL Brands]は30日間アーカイブされ、その期間内に復元できます。

## ブランドガイドライン

GenStudio for Performance Marketingのガイドラインは、ブランド独自のアプローチ、トーン、ビジュアルフィールなどを明確に示します。

>[!NOTE]
>
>英語以外の言語で作成されたブランドガイドラインをアップロードするか、英語以外の言語を使用して手作業でブランドを作成すると、GenStudio for Performance Marketingには同じ言語でブランドガイドラインが表示されます。

[&#x200B; プラットフォームガイドラインおよびADA アクセシビリティ標準](/help/user-guide/guidelines/overview.md#compliance)から抽象化された一部のチャネルガイドラインは、デフォルト値として事前に入力されます。

![&#x200B; ブランドガイドライン &#x200B;](/help/assets/brands.png){width="600" zoomable="yes"}

以下のガイドラインの節では、各ガイドラインのカテゴリ、例のガイドライン、および潜在的な結果について詳しく説明します。 ガイドラインの例は、結果出力を形作るサンプル入力です（以下の結果例）。これを使用して、独自のブランドガイドラインを作成します。

| ------------------| :---------- | :---------- |
| [!UICONTROL 声のトーン &#x200B;] | 書面によるコミュニケーションで表されたブランドの個性 | 説明的な単語を使用して、ブランドの感情とトーンを明確に伝える。 <br>_例_:「本物」、「遊び心」 <br><br>簡潔な（3 ～ 6つのガイドライン）を作成し、主要なブランドボイス属性に焦点を当てます。<br><br>一貫したブランドアイデンティティを構築するために、コミュニケーションチャネル全体で一貫した単語やフレーズを選択します。 |

### ロゴ

[!DNL Brand]個のロゴは、企業のアイデンティティ、価値観、製品を表す視覚的なシンボルとして機能します。 JPG、JPEG、SVG、またはPNG形式で[!DNL Brand]の1つまたは複数のロゴ（ファイルごとに1 GBを超えないこと）をアップロードできます。 ロゴは、[!DNL Brand]の作成中に、または個々の&#x200B;_[!DNL Brands]ビュー_&#x200B;から追加できます。 ロゴは、特定の[!DNL Brand]のサムネール画像として使用できます。

アップロードされた各ロゴには、ロゴタイル内に次の詳細が含まれています。

- ファイルタイプ
- ファイルサイズ
- 解決策

ロゴのアクションメニュー（3つのドット）から、個々のロゴの詳細を[表示および管理](/help/user-guide/guidelines/add-guidelines.md#manage-brands)できます。

- **[!UICONTROL 詳細を表示]**
- **[!UICONTROL ダウンロード]**
- [**[!UICONTROL &#x200B; ブランドのサムネールとして使用]**](/help/user-guide/guidelines/add-guidelines.md#change-brand-thumbnail)
- **[!UICONTROL 名前を変更]**
- **[!UICONTROL 削除]**

### カラー

個別の[!DNL Brands]内のカラーを管理して、一貫性のあるアクセス可能なテキストのスタイル設定を行います。 16進数コードまたはRGB コードでカラーを選択または定義し、単色のみをサポートします（不透明度はサポートされていません）。 無制限のカラーを追加し、ギャラリー形式で表示し、16進数またはRGB コードから人間が読みやすい名前に変更できます。

![&#x200B; ブランドカラー](/help/assets/colors.png){width="600" zoomable="yes"}

カラーのアクションメニュー（3つのドット）から、カラーを&#x200B;**[!UICONTROL 編集]**、**[!UICONTROL 名前を変更]**&#x200B;または&#x200B;**[!UICONTROL 削除]**&#x200B;できます。

## ブランド検証

GenStudio for Performance Marketingでは、ブランド検証が、[!DNL Brands]、[!DNL Products]、[!DNL Personas]の生成AI機能と連携して重要な役割を果たします。 これにより、あらゆるコンテンツが、ブランドアイデンティティとアクセシビリティに関する懸念との一貫性を維持できます。

[&#x200B; ブランド検証](/help/user-guide/guidelines/brand-validation.md)を参照してください。
