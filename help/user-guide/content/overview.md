# Adobe GenStudio for Performance Marketing [!DNL Content]

GenStudio for Performance Marketing [!DNL Content]では、すべてのブランド承認済みアセット、エクスペリエンス、テンプレートを一元的に保存、検索、共有できます。 生成AIを利用すれば、コンテンツの編集および再構成、コンテンツの利用とセンチメントに関するinsightの活用、既存のアセットの更新および再利用が可能になります。

## [!DNL Content] 機能

GenStudio for Performance Marketing [!DNL Content]を使用すると、承認済みアセットを簡単に管理し、コンテンツのライフサイクルを高速化できます。 **[!UICONTROL [!DNL Content]]**&#x200B;では、パフォーマンスの高いコンテンツを検索、検索、再利用したり、アセットやエクスペリエンスを選択して、常緑キャンペーンの新しいバリエーション作成を促進したりできます。

### アセットのアップロード

**[!UICONTROL アセットを追加]** パネルでは、ローカルファイル内のアセットをドラッグ&amp;ドロップまたは参照できます。 必要に応じて、OneDriveやDropboxなどの外部ソースからアセットを読み込むことができます。 [!UICONTROL &#x200B; キャンペーン &#x200B;]またはプロジェクト名は、アップロード前に必要な唯一の詳細です。 キーワードなどの詳細情報を提供することで、チームの全員が関連コンテンツを迅速に特定し、取得できるようになります。

### 承認済みコンテンツの検索と発見

承認されたアセットを保存すると、[!DNL Content]は豊富なメタデータ、生成コンテキスト、監査履歴を保持します。 アセットの詳細は、フィルターと検索文字列を使用して検索結果をすばやく絞り込むのに役立ちます。 GenStudio for Performance MarketingのAIは、色などの特定の詳細を自動的に識別して保存します。 検索フィールドで`blue`を検索すると、コンテンツリストが青のタグ付きのアセットに絞り込まれます。

### テンプレートの操作

テンプレートを利用することで制作プロセスが高速化され、複数のチャネルをまたいで一貫性のあるコンテンツを生成できます。 カスタムテンプレートをアップロードし、事前設定されたレイアウトやデザイン要素を使用して、制作プロセスのベースとして使用できます。 [&#x200B; テンプレートの操作](../templates/use-templates.md)の基本的なテンプレート要素と手順について説明します。 [&#x200B; テンプレートのカスタマイズ &#x200B;](../templates/customize-template.md) トピックを使用したテンプレートの構築について詳しく説明します。

### 承認済みエクスペリエンスの翻訳

承認された電子メールや有料メディアのエクスペリエンスを、[!DNL Content] _[!UICONTROL エクスペリエンス]_ ギャラリーまたは[!DNL Create] ランディングページから直接複数の言語に翻訳できます。 [&#x200B; コンテンツから翻訳](../create/translate-experiences.md#translate-from-content)または[作成から翻訳](../create/translate-experiences.md#translate-from-create)を参照してください。
