# コネクテッド TV体験の構築

[!DNL GenStudio for Performance Marketing]の[[!DNL Create]](/help/user-guide/create/overview.md)を使用して、コネクテッド TV （CTV）広告を1か所で作成できます。概要や共有ガイドラインから、生成、シーンベースの絞り込み、承認、パブリッシャー向けの書き出しまで、様々な方法があります。 以下のワークフローは完全に[!DNL GenStudio for Performance Marketing]で実行されます。個別のCTV アプリまたはパブリッシャーの埋め込みはありません。

## 前提条件

CTV広告を作成する前に、次の点を確認してください。

* [!DNL GenStudio for Performance Marketing] にアクセスします。
* **[!DNL Brands]**、**[!DNL Products]**&#x200B;および&#x200B;**[!DNL Personas]**&#x200B;は、[!DNL GenStudio for Performance Marketing]で共有オブジェクトとして設定されています。 これらのオブジェクトが生成にどのように影響するかについては、[&#x200B; ガイドラインの概要](/help/user-guide/guidelines/overview.md)を参照してください。
* キャンペーンアセット（ビデオクリップ、画像、ロゴ、音楽）は推奨されますが、必須ではありません。生成AIは、アセットが不足しているか、不完全な場合にギャップを埋めることができます。

## 新しいCTV広告の作成

このワークフロー内のすべては[!DNL GenStudio for Performance Marketing]内で実行されます。

![&#x200B; コネクテッド TVの作成カード](./ctv-tile.png){width="50%"}
**CTV作成に移動するには**:

1. [!DNL GenStudio for Performance Marketing]にログインします。
1. ホームまたは作成サーフェスから、**[!UICONTROL 作成]**&#x200B;に移動します。
1. CTV作成カードを使用して&#x200B;**CTV**&#x200B;を選択します。
1. 「**[!UICONTROL CTV広告を作成]**」をクリックします。

単一の合理化されたCTV制作体験が構築されます。 最初に広告タイプを選択する必要はありません。

## 概要の設定

ブリーフと入力が、広告の生成方法となります。 広告を生成するプロセスのコンテキストと制約を提供するチャンスです。

![概要を設定](./ctv-brief.png){width=80%&quot; align=&quot;center&quot;}

**概要を設定するには**:

1. 既存の共有オブジェクトから&#x200B;**[!DNL Brands]**、**[!DNL Products]**&#x200B;および&#x200B;**[!DNL Personas]**&#x200B;を選択します。
1. **クリエイティブ概要**&#x200B;を直接入力するか、アップロードして追加します。 キャンペーンの目的、主要メッセージ、制約を含めます。
1. **広告期間**&#x200B;を15秒または30秒に設定します。
1. オプションで&#x200B;**assets**&#x200B;を追加します。 ビデオクリップ、画像、ロゴ、音楽、ナレーション、またはイントロ/アウトロカード（ドラッグ&amp;ドロップまたはファイル選択）をアップロードするか、[!DNL Content] リポジトリからアセットを選択します。
1. 「**[!UICONTROL 生成]** ボタンをクリックします。

アセットが見つからないか不完全な場合、[!DNL GenStudio for Performance Marketing]はAIを使用して、見つからないシーン、音楽、ナレーションを生成できます。 お客様が提供するAssetsは、生成されたマテリアルよりも常に優先されます。

自動的に[!DNL GenStudio for Performance Marketing]:

* **[!DNL Brands]**、**[!DNL Products]**、**[!DNL Personas]**&#x200B;のコンテキストと共に概要を解釈します。
* 完全なCTV広告構造を構築します。
* 必要に応じて、シーン、テキストオーバーレイ、音楽、ナレーションを作成します。
* CTVに準拠したデュレーションとフォーマットを適用します。

その結果、ドラフトのないタイムラインではなく、詳細に構築されたプレビュー可能なCTV広告を実現できます。

## 広告の編集と調整

シーンベースのエディターを使用すると、すべてを再生成せずに広告を微調整できます。

シーン ストリップ内のシーンをクリックして、編集のために開きます。 次のような編集を実行できます。

* AIを使用して、単一のシーンを置き換えたり、再生成したりします。
* シーン プロンプトを編集して、バリエーションを作成します。
* シーンの並べ替えまたはトリミング。
* テキストオーバーレイを編集します。
* 音楽やナレーションを入れ替えたり、ミュートしたり、置き換えたりできます。
* シーン間のトランジションを調整する

編集の範囲が設定されているので、一度に1つのシーンを再生成して、より迅速な反復とクリエイティブの更新を実現できます。

>[!NOTE]
>
>エディターでは、ビデオクリップ内のオブジェクト *の変更はサポートされていません（例えば、アイテムの削除、製品カラーの変更、人物の表示方法の変更など）。*

## レビューと承認

統合された承認ワークフローを使用して、ブランドレビュー用に広告を送信します。 ブランドおよび関係者のレビュー担当者は、メッセージ、ビジュアル、ブランドコンプライアンスをチェックします。 承認者は広告を検証します。マーケターの代わりにビデオ編集を実行することは期待されていません。

## 書き出し

承認後、次のことができます。

* 最終的なCTV広告を、パブリッシャーに対応したコンプライアンス形式で書き出します。
* 広告を[!DNL Content]に保存し直してください。
* 下流のCTVの購入および人身売買ワークフローで使用します。

Creativeは、再エンコーディングや手戻りなしでアクティベーションできる状態を意図しています。
