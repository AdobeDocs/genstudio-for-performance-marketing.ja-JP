# Adobe GenStudio for Performance Marketing Beta リリースノート

これらのメモは、10月4日に終了する週のAdobe GenStudio for Performance Marketingの重要な修正と機能強化を示しています。

## 新機能と機能強化

* 製品全体のフィルタリング機能が強化されました。 [!DNL Insights]の年齢と性別によるフィルタリングに関する問題が解決されました。 <!-- GS-1198 -->

* Adobe Expressを使用して、画像アセット（JPGまたはPNG）を直接編集できます。 コンテンツエディターは、_[!UICONTROL Adobe Expressを活用した]_ Canvasを使用して、GenStudio for Performance Marketingを離れることなく、背景の削除、生成塗りつぶしの適用、エフェクトの調整、画像の切り抜きを行うことができます。<!-- GS-4615 -->

* 生成されたバリエーション、生成されたメール、コンテクストに即したメッセージのプログレッシブ読み込み体験が向上しました。<!-- GS-4651 3062-->

* コンテンツエディターは、切り抜き調整機能を使用して、レンダリングされた画像をバリエーションで切り抜くことができるようになりました。<!-- GS-2342 -->

* テンプレートのサイズ変更と複製に関する問題が解決されました。<!-- GS-4895 -->

* ブランド検証で、検証中に発生するエラーの原因が説明されるようになりました。

* テンプレートプレビューに、画像テキストが期待どおりに表示されるようになりました。<!-- GS-5917 -->

## その他の機能強化と修正された問題

* [!DNL Create] キャンバスは、テンプレートからカスタムフォントを期待どおりにレンダリングするようになりました。<!-- GS-3415 -->

* Metaと書き出し中に、正しいフォントが適用されるようになりました。<!-- GS-5875 -->

* テンプレートのアップロードに関する問題により、アップロードは成功しましたが、製品インターフェイスで表示されない問題は解決されました。<!-- GS-4815 5650-->

* サイズを変更した後、Meta広告を手動でトリミングできるようになりました。<!-- GS-5871 -->

* ユーザーは、Facebookにもログインしている場合に、チャネルのMeta広告アカウントに2回ログインする必要がなくなりました。<!-- GS-3009 -->

* アセットとエクスペリエンスのカンバスビューが、コンテンツ作成、レビュー、承認プロセス全体で一貫性を保つようになりました。<!-- GS-5877 -->

* サイズ変更操作後に再生成する際に、カンバスに4つのバリエーションのみが表示されるようになりました。<!-- GS-5869 -->

* ブラウザーベースのスペルチェックが、[!DNL Create] Canvasで期待どおりに機能するようになりました。<!-- GS-5760 -->

* **[!UICONTROL PNGとして書き出しを選択すると、ディスプレイ広告がPNG ファイルとして書き出されるようになりました。]** 以前は、PNG形式を選択すると、ディスプレイ広告はJPEGとして書き出されていました。<!-- GS-5545 -->

* **[!UICONTROL 手動切り抜き]** ボタンと&#x200B;**[!UICONTROL 生成]** ボタンの間でパディングが増加しました。 以前は、**[!UICONTROL 手動切り抜き]** ボタンが部分的に隠されていました。<!-- GS-6084 -->

* テンプレートプレビューに、Google フォントが期待どおりに表示されるようになりました。<!-- GS-5946 -->

* 読み込まれたTypeKit フォントとGoogle フォントが、書き出し中に正常に読み込まれるようになりました。<!-- GS-5948 -->

* カスタムテンプレートでコンテンツを生成する際の問題を解決しました。 以前、コンテンツエディターがカスタムテンプレートを使用してアセットを生成しようとすると、生成ポップアップは表示されず、コンソールにエラーが表示されていました。<!-- GS-5262 -->

* コンテキストメニューから左クリックする前にユーザーがカンバスを右クリックすると、DisplayAds ドラフトカンバスの位置が維持されるようになりました。 以前は、ユーザーが左クリックしたときにカンバスが移動し、ドラフトコンテンツが部分的にアクセスできなくなっていました。 <!-- GS-5687 -->

* 画像の再生成が完了するまで、シマーエフェクトの読み込みが保持されるようになりました。 <!-- GS-5811 -->

* ユーザーが生成された電子メール、Meta広告、ディスプレイ広告を編集した後も、ブランド検証スコアが無効にならなくなりました。 以前は、このスコアは非表示でした。<!-- GS-5379 -->

* CSS スタイルが`body`要素にアタッチされたテンプレートは、エクスペリエンスの書き出し中に期待どおりに活用されるようになりました。<!-- GS-5947 -->

* サイズの大きい画像を手動で切り抜く際の問題を修正しました。<!-- GS-6039 -->

* ユーザーが[!DNL Content]に新しいアセットを追加すると、1つのポップアップメッセージのみが表示されるようになりました。<!-- GS-5020 -->

* テキスト編集時のカンバスのパフォーマンスが向上しました。 <!-- GS-5118 -->

* [!DNL Create]電子メールまたはMetaおよびCanvasの文字列間にスペースが見つからない問題を修正しました。<!-- GS-5019 -->

* Expressで編集後、特殊文字を含む名前のファイルを保存できるようになりました。<!-- GS-6131 -->

### ローカライゼーション

このリリースでは、次のインターフェイス領域を含め、製品インターフェイス全体でのローカライゼーションが改善されています。

* [!DNL Create] プロンプトメニューの&#x200B;**[!UICONTROL 詳細情報]** オプションの宛先のURL。<!-- GS-5029 -->

* [!DNL Insights] > [!DNL Experience]の検索入力フィールドに隣接する数値形式。<!-- GS-4494 -->

## 既知の問題

* 再生成された電子メールフラグメントは、選択後にバリエーションに表示されません。 （ただし、ドラフトが再度開かれた後にバリエーションが表示されます） <!-- GS-5913 -->
