# キャンペーンの概要

[!DNL Insights] _[!UICONTROL キャンペーン]_ ビューには、接続されたチャネルとアカウントのキャンペーンのリストが表示されます。

{{connect-insights}}

_[!UICONTROL キャンペーン]_ テーブルは、チャネルベースの広告キャンペーンを使用して整理されます。 表の右側にある設定（歯車）アイコンをクリックして、表示可能な列を切り替えます。

![&#x200B; キャンペーンのフィルターとテーブル &#x200B;](/help/assets/insights-campaigns-filter.png){zoomable="yes"}

キャンペーンを選択すると、[[!UICONTROL 広告] タブ &#x200B;](published-experiences.md)が開き、そのキャンペーンに関連付けられている広告名のリストが表示されます。これにより、全体的な成功に貢献した広告を特定できます。

{{filter-table}}

## 目的

Meta Adsを使用したキャンペーンの作成時に、ビジネス目標に沿った目標を選択した可能性があります。 GenStudio for Performance Marketingで表示されるMeta広告の6つの目的があります。

1. **認知度**：最大のオーディエンスにリーチし、ビジネスに認知度をもたらします。
1. **トラフィック**：サイトまたはアプリケーションへのトラフィックを増やします。
1. **エンゲージメント**：既存顧客と潜在顧客のインタラクション。
1. **リード**：オーディエンスを拡大するための接続を構築します。
1. **アプリのプロモーション**：アプリケーションを宣伝します。
1. **Sales**：主に製品を使用する可能性の高いユーザーにリーチすることに重点を置きます。

## チャネルパフォーマンス

パフォーマンス目標に応じて、[!DNL Insights]指標を使用すると、目標を達成しているかどうかを評価できます。

例えば、認知度が目的である場合、`impressions`の割合を増加させることは、リーチを拡大していることを示している可能性があります。 それが目的を達成しているかどうかを把握するには、`clicks`や`video plays`など、コンテンツが魅力的であることを示す指標を確認できます。 オーディエンスは、どの程度効果的にコンテンツに接しているか？

次の表に、チャネルビューの主要なデジタルマーケティング指標の定義とインサイトを示します。 各指標には、チャネルに関連する概要、指標の計算方法、マーケティングキャンペーンに対する重要性と影響を把握するのに役立つ1つ以上のインサイトが含まれています。

| 指標 | 定義 | Insight |
| ----------- | ----------------------------- | -------------------------------- |
| **[!UICONTROL Meta キャンペーン名]** | 接続されたチャネルアカウントのキャンペーン名のリスト。 キャンペーンのステータスは、`Active`、`Paused`、`Deleted`または`Archived`です。 ステータスと目的ごとにキャンペーンをフィルタリング。 | いずれかの主要指標の列見出しをクリックして、テーブルを並べ替えます。 |
| **[!UICONTROL 起動]** | キャンペーンがリリースされたか、市場に公開された日付。 | インプレッション数が多い場合は、広告が目的のオーディエンスにリーチしていることを示している可能性があります。 |
| **[!UICONTROL インプレッション]** | インタラクションや視聴に関係なく、コンテンツがチャネル内で読み込まれるたびにカウントします。 | インプレッション数が多い場合、オーディエンスの幅広い可視性を把握できます。しかし、Adobe insightの真のパフォーマンスを把握するためには、他のエンゲージメント指標と組み合わせて検討する必要があります。 |
| **[!UICONTROL クリック]** | ユーザーが広告キャンペーン内でクリック可能な要素（リンクやcall-to-action ボタンなど）を操作した回数。 | クリック数が多い場合は、コンテンツへの関心とエンゲージメントが高いことを示します。これは、効果的で適切なオーディエンスにリーチできる可能性があります。 |
| **[!UICONTROL CTR &#x200B;]**<br>_クリック率_ | チャネル内でキャンペーンクリックが発生したインプレッションの割合（%）。<br>**計算**: （`clicks` x `impressions`） x 100 = % | クリック率が高い場合、コンテンツがメッセージやデザインにおいてオーディエンスにとって非常に関連性が高く、やる気を喚起し、オーディエンスの興味を効果的に惹きつけていることを示しています。 |
| **[!UICONTROL CPM &#x200B;]**<br>_千件当たりのコスト_ | 広告キャンペーンの広告インプレッション数が1,000件ごとに費用がかかります。 <br>**計算**: （合計金額`spent`を`impressions`で割った値） x 1000 | クリックスルー率が高い場合は特に、値が低い場合は、費用対効果の高い表示が示される可能性があります。 |
| **[!UICONTROL CPC &#x200B;]**<br>_クリックあたりのコスト_ | 広告キャンペーンの各クリックに関連付けられた平均コスト。<br>**計算**：合計金額`spent`を`clicks` = $で割った | 特にコンバージョンの増加と比較すると、平均コストが低いことは、広告費がコスト効率に優れていることを示している可能性があります。 |
| **[!UICONTROL 動画を再生]** | リプレイを除く、広告キャンペーンでビデオが再生され始めた回数。 ユーザーが「再生」をクリックするか、ビデオが自動再生されると、ビデオの再生が開始されます。 | 動画再生数が多い場合、動画が注目を集めていることを示している可能性があります。また、サムネール、タイトル、配置が効果的に視聴者を引き付けている可能性もあります。 |
| **[!UICONTROL 予算]** | キャンペーン目標を達成するために広告キャンペーンに割り当てられた総資金。 | 予算が多ければ多いほど、より広範なリーチのためのリソースが増え、潜在的により大きな影響が及ぶ可能性があります。 |
| **[!UICONTROL 支出]** | 異なるプラットフォーム間で広告を配置するために、一定期間に予算から費やした金額。 | 短期間で多くの費用が発生した場合、迅速な使用を示している可能性があり、リソースの早期枯渇につながる可能性があります。 また、主要なパフォーマンス指標に対して費やした費用を追跡し、全体的な投資収益率を監視できます。 |
