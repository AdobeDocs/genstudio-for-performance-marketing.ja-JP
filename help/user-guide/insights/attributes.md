# 属性の概要

[!DNL Insights] _[!UICONTROL 属性]_ ビューには、選択したチャネルアカウントの広告キャンペーンで使用される属性のリストが表示されます。

{{connect-insights}}

_[!UICONTROL 属性]_ テーブルは、[!UICONTROL 属性]名を使用して整理されています。 **[!UICONTROL 画像]** ボタンと&#x200B;**[!UICONTROL ビデオ]** ボタンを使用して、リストタイプを切り替えることができます。 表の右側にある設定（歯車）アイコンをクリックして、表示可能な列を切り替えます。

![属性フィルターとテーブル &#x200B;](/help/assets/insights-attributes-filter.png){zoomable="yes"}

{{filter-table}}

## 属性の詳細

属性は、色、構図、視覚要素、その他のプロパティなどの固有の詳細によって、[広告](published-experiences.md#ad-details)と[&#x200B; メディア &#x200B;](media.md#media-details)を識別するのに役立ちます。

属性の詳細ビューでは、選択した属性を使用する広告を確認できます。 詳細には、属性の合計パフォーマンスと、各広告に関連するパフォーマンス指標の内訳が含まれます。

![属性パフォーマンス指標](/help/assets/insights-attribute-details.png){zoomable="yes"}

GenStudio for Performance Marketingは、特定の機能を検出し、メディアコンテンツまたは広告に適切な属性をタグとして適用します。 これらのタグの例については、[&#x200B; カテゴリ &#x200B;](#categories)を参照してください。 広告に関連付けられたすべての属性を表示するには、テーブルの右側にある設定（歯車）アイコンをクリックして、**[!UICONTROL 属性]**&#x200B;列を選択します。

## カテゴリ

属性&#x200B;_カテゴリ_&#x200B;は、共通の特徴を共有する関連属性を整理する分類グループです。 これらのカテゴリは、より大きなコンテキストを提供し、その適用と使用を促進することで、特定の属性の発見、識別、理解を合理化するのに役立ちます。

GenStudio for Performance Marketingは、AdobeのAIとマシンラーニング機能を使用して、[画像](image-features.md)、[動画](video-features.md)、[&#x200B; テキスト &#x200B;](text-features.md)を調査し、正確性の可能性に基づいて広告とメディアに属性を適用します。

広告およびメディアコンテンツの検出済み属性リストは完全ではありません。 豊富な機能セットを含むコンテンツは、AIが特定する3つの主要な機能に制限される場合があります。 例えば、次の図には、複数のオブジェクト、描画色、背景色など、複数の検出済み画像属性が含まれています。

![画像属性](/help/assets/category/asset-attributes.png "Toucanの画像に、複数の検出属性"){width="300" zoomable="yes"}が含まれています

>[!INFO]
>
>検出され、自動的に適用されるタグは編集できません。

## 属性のパフォーマンス

インサイト指標は、どの属性がより多くの顧客エンゲージメントを促すのかを評価するのに役立ちます。

次の表は、[!UICONTROL 属性] テーブルビューの主要なデジタルマーケティング指標の定義とインサイトを示しています。 各指標には、属性に関連する概要、指標の計算方法、広告キャンペーンに対する重要性と影響を把握するのに役立つ1つ以上のインサイトが含まれています。

| 指標 | 定義 | Insight |
| ---------------------- | ----------------------------- | -------------------------------- |
| **[!UICONTROL 属性]** | 属性名。 | いずれかの主要指標の列見出しをクリックして、テーブルを並べ替えます。 |
| **[!UICONTROL カテゴリ]** | 属性の固有の品質を表す[&#x200B; カテゴリ &#x200B;](#categories)です。 |  |
| **[!UICONTROL #枚の画像]** | この属性を持つ画像の数。 | 属性テーブルのカウントは、属性の詳細ビューのカウントとは異なる場合があります。 この不一致は、MetaやGenStudioなどのチャネルソースでサマリー計算が若干異なる場合に発生する可能性があります。 |
| **[!UICONTROL #本の動画]** | この属性を持つビデオの数。 | 属性テーブルのカウントは、属性の詳細ビューのカウントとは異なる場合があります。 この不一致は、MetaやGenStudioなどのチャネルソースでサマリー計算が若干異なる場合に発生する可能性があります。 |
| **[!UICONTROL インプレッション]** | インタラクションや表示に関係なく、この属性を持つ画像またはビデオがチャネルに読み込まれるたびにカウントされます。 | インプレッション数が多い場合、幅広い可視性が示されます。ただし、Adobe insightの真のパフォーマンスを把握するには、他のエンゲージメント指標と比較してインプレッション数を考慮する必要があります。 |
| **[!UICONTROL クリック]** | ユーザーがこの属性を使用して画像またはビデオを操作した回数。 | クリック数が多い場合は、コンテンツへの関心とエンゲージメントが高いことを示します。これは、効果的で適切なオーディエンスにリーチできる可能性があります。 |
| **[!UICONTROL CTR &#x200B;]**<br>_クリック率_ | この属性を持つ画像またはビデオのクリックにつながったインプレッションの割合（%）。<br>**計算**: `clicks`を`impressions`で割った | クリック率が高い場合、コンテンツがメッセージやデザインにおいてオーディエンスにとって非常に関連性が高く、やる気を喚起し、オーディエンスの興味を効果的に惹きつけていることを示しています。 |
| **[!UICONTROL CPM &#x200B;]**<br>_千件当たりのコスト_ | この属性を持つ画像またはビデオの1,000個の広告インプレッションごとに費用が発生します。<br>**計算**：合計金額`spent`をリーチで割り、その後1000を掛けます | クリックスルー率が高い場合は特に、値が低い場合は、費用対効果の高い表示が示される可能性があります。 |
| **[!UICONTROL CPA &#x200B;]**<br>_アクションあたりのコスト_ | 購入やサブスクリプションなど、特定の顧客アクションを達成するために費やした平均コスト。<br>**計算**：合計金額`spent`を完了した顧客アクション数で割った値 | 顧客の行動を促進する属性を特定する： |
| **[!UICONTROL CPC &#x200B;]**<br>_クリックあたりのコスト_ | この属性を持つ画像またはビデオをクリックするたびに関連付けられる平均コスト。<br>**計算**：合計金額`spent`を`clicks`で割った値 | 特にコンバージョンの増加と比較すると、平均コストが低いことは、広告費がコスト効率に優れていることを示している可能性があります。 |
| **[!UICONTROL 支出]** | 特定の期間における属性に関連する予算から費やした金額。 | 短期間で多くの費用が発生した場合、迅速な使用を示している可能性があり、リソースの早期枯渇につながる可能性があります。 また、主要なパフォーマンス指標に対して費やした費用を追跡し、全体的な投資収益率を監視できます。 |
