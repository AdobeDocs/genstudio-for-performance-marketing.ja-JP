# Adobe GenStudio for Performance Marketingのインサイト

Adobe GenStudio for Performance Marketing [!DNL Insights]は、データ主導の意思決定に役立つ、コンテンツのパフォーマンスに関する高度な分析機能とインサイトを提供します。

[!DNL Insights] ダッシュボードでは、次の操作を実行できます。

- **最も効果的なコンテンツを特定**：様々なオーディエンスに対してどのコンテンツが最も効果的かを特定し、今後のコンテンツやキャンペーンの傾向をトレンドの好みに合わせて調整します。
- **パフォーマンスの低いコンテンツを最適化**：パフォーマンスの低いコンテンツを見つけ出し、統合された生成AIを使用してバリエーションをすばやく作成し、ゼロから始めずに効果を向上させる可能性があります。
- **パフォーマンスの高いコンテンツの活性化**：成功したコンテンツを取り込み、オーディエンスの広告を更新するか、ヒーローコンテンツを新しいキャンペーンで使用するように調整することで、ライフサイクルとパフォーマンスが拡張される可能性があります。

[!DNL Insights] モジュールには、**[!UICONTROL Insights 2.0]**&#x200B;が含まれています。これは、有料ソーシャル広告のクロスチャネルパフォーマンスエクスペリエンスです。 この記事の[&#x200B; ダッシュボード &#x200B;](#dashboard) セクションの詳細なテーブルとギャラリーのビューと一緒に動作します。

## インサイト 2.0 {#insights-20}

**[!UICONTROL Insights 2.0]**&#x200B;は、接続されたアカウントをまたいで有料ソーシャルマーケティングのパフォーマンスを明確に把握できるパフォーマンスインテリジェンスレイヤーをマーケターに提供します。

**[!UICONTROL Insights 2.0]では、次の操作を実行できます：**

- **クロスチャネルまたはシングルチャネルの概要を確認（MetaおよびLinkedIn）**：両方の有料ソーシャルチャネルで統合されたスナップショットを確認するか、1つのチャネルにドリルダウンします。
- **クロスチャネルパフォーマンスレポートを使用**：各チャネルの結果のシェアを、総支出（パーセンテージと金額）およびCTR、CPC、CPMなどのパフォーマンスシェア指標を含む、貢献度の可視化で表示します。
  ![&#x200B; クロスチャネルパフォーマンスレポート &#x200B;](./cross-channel-report.png)
- **広告パフォーマンスレポートを使用**：最適化の決定をサポートするランキングと指標を使用して、パフォーマンスの高い広告と低い広告を特定します。
  ![広告パフォーマンスレポート &#x200B;](./ad-performance-report.png)
- **Meta コンバージョン指標を分析**: funnelの各段階（エンゲージメント済みの訪問数、情報請求、アプリの起動数、見込客、アプリの完了数など）をまたいでCPAを可視化し、コンバージョンの傾向を長期的に追跡します。コンバージョンデータはGenStudio for Performance Marketingで利用できます。
  ![Meta コンバージョン指標](./meta-conversion-report.png)
- **広告タグからインサイトを探る**：広告トラッキング IDは構造化タグに解析されるので、定義したディメンション（call to action、地域、フォーマット、コンセプトなど）ごとにパフォーマンスを分析し、それらのディメンション間の予算配分を確認して、手動で命名規則をデコードする時間を短縮できます。
  ![&#x200B; インサイト 2.0広告タグレポート &#x200B;](./ad-tags-insights.png)

>[!NOTE]
>
>**[!UICONTROL インサイト 2.0]**&#x200B;には、現在&#x200B;**Meta**&#x200B;と&#x200B;**LinkedIn**&#x200B;のみが含まれています。 現時点では、TikTok、DV360およびInnovidは&#x200B;**[!UICONTROL Insights 2.0]**&#x200B;の概要に含まれていません。 [&#x200B; ダッシュボード &#x200B;](#dashboard) セクションの&#x200B;**[!UICONTROL キャンペーン]**、**[!UICONTROL 広告]**、**[!UICONTROL メディア]**&#x200B;および&#x200B;**[!UICONTROL 属性]** ビューでは、[&#x200B; サポートされるチャネル &#x200B;](#channels-supported)に記載されている広範なチャネルセットを引き続きサポートします。

## データコネクタ

[!DNL Insights]を初めて開いたときに、Adobe GenStudio for Performance Marketingとチャネルアカウントを接続するためのバナーが表示される場合があります。

この連携により、GenStudio for Performance Marketingはアクティブなマーケティングキャンペーン、メディア、広告から統計データを受け取ることができます。 GenStudio for Performance Marketingでは、まず、過去6 ヶ月間のデータを収集し、最新のデータを分析してアクションを起こすためのツールを備えています。

{{connect-insights}}

## サポートされているチャネル {#channels-supported}

インサイトでサポートされるチャネルには、Meta、LinkedIn、TikTok、DV360、Innovidなどがあります。

Meta、LinkedIn、TikTokにより、キャンペーン、広告、メディア、属性を包括的に可視化できます。 DV360とInnovidは現在、より限定的なデータ範囲を提供しています。

現時点では、DV360およびInnovidではメディアデータは使用できません。つまり、「属性」タブもこれらのチャネルには表示されません。 「属性」タブは、エクスペリエンスから抽出された特性を表示するために、メディアレベルのデータに依存します。

この制限は、有料メディアプラットフォーム自体の制約によるものであり、GenStudio for Performance Marketingの問題ではありません。

## ダッシュボード {#dashboard}

[!DNL Insights] ダッシュボードには、コンテンツタイプごとに設定可能なテーブルがあります：[!UICONTROL &#x200B; チャネル &#x200B;]、[!UICONTROL 広告]、[!UICONTROL &#x200B; メディア &#x200B;]、および[!UICONTROL 属性]。

![[!DNL Insights] ダッシュボード &#x200B;](/help/assets/insights-dashboard.png)

各ビューには対応するテーブルが表示され、キーワード、フィルタリング、日付範囲で検索できます。 表の右側にある設定（歯車）アイコンをクリックして、表示可能な列タイプを切り替えることができます。 _[!UICONTROL 概要]_&#x200B;行には、列の合計または平均が表示される場合があります。

[!UICONTROL 広告]、[!UICONTROL &#x200B; メディア &#x200B;]および[!UICONTROL 属性]には、画像またはビデオのサムネールを含むカードを使用してアセットをスキャンおよび並べ替えることができるギャラリービューが含まれています。 各カードに3つの主要指標のいずれかを表示するオプションがあります：`Click-through rate`、`Cost per click`、`Spend`。

### キャンペーン

[[!DNL Insights] _[!UICONTROL &#x200B; キャンペーン &#x200B;]_&#x200B;ビュー](campaigns.md)はデフォルトのビューで、目的、予算、開始日、アクティビティなど、アクティブなキャンペーンの詳細のリストが表示されます。 GenStudio for Performance Marketingが統計データの受信を開始するように、チャネルアカウント [&#128279;](/help/user-guide/connectors/connect-channel.md)を接続してください。

### 公開されたエクスペリエンス

[[!DNL Insights] _[!UICONTROL 公開されたエクスペリエンスの詳細&#x200B;]_&#x200B;ビュー](published-experiences.md)は、エクスペリエンスの有効性を評価することに重点を置いています。 [!UICONTROL 公開されたエクスペリエンス &#x200B;] ビューでは、指定された日付範囲内での配置に基づいて、エクスペリエンスの指標を分析できます。_[!UICONTROL &#x200B; エクスペリエンス名&#x200B;]_&#x200B;をクリックすると、エクスペリエンスのパフォーマンス指標、プレースメントによるパフォーマンス、属性を表示できます。

### メディア

[[!DNL Insights] _[!UICONTROL &#x200B; メディア &#x200B;]_&#x200B;ビュー](media.md)は、クリエイティブ コンテンツのパフォーマンスを分析するために設計されています。 クリック数やインプレッション数など、選択した指標の改善に貢献するメディア属性を特定できます。

メディアコンテンツをクリックすると、様々な広告や広告配置におけるパフォーマンスに関する詳細なコンテキストが得られます。

![&#x200B; メディアの詳細](/help/assets/insights-media-details.png){width="600" zoomable="yes"}

メディアの詳細ビューでは、左側にアセットのサムネールと属性のリストが表示されます。 ハイライトされた指標は3つあります：`Click-through rate`、`Cost per click`、`Spend`。 パフォーマンスのハイライトは、選択した期間の平均値（点線）と実際の値（実線）の比較を示します（デフォルトは`Last 30 days`）。

### 属性

メディア _属性_&#x200B;は、色、トーン、構図（被写体、フォント、ビジュアル要素など）、その他の主要コンポーネントなどの固有の詳細によって、クリエイティブなコンテンツを識別するのに役立ちます。 属性は、多くの場合、最も測定および分析されていないコンテンツ情報のセットです。

[[!DNL Insights] _[!UICONTROL 属性&#x200B;]_&#x200B;ビュー](attributes.md)は、特定のオーディエンス、チャネル、地域でパフォーマンスが高い属性を調査および特定するのに役立ち、季節的な傾向を強調するのに役立ちます。 これらのインサイトを活用すれば、パフォーマンスの高い属性を使用してバリエーションを作成したり、特定のオーディエンスをターゲティングしたり、さまざまなキャンペーン戦略を試したりできます。

### 広告タグ

[[!DNL Insights] _[!UICONTROL 広告タグ &#x200B;]_&#x200B;ビュー](ad-tags.md)には、接続されたチャネルとアカウントの広告のリストが表示されます。_ ad _は、マーケティングキャンペーンの一環として特定のオーディエンスに配信することを目的とした、視覚的でインタラクティブなコンテンツを含むプロモーションアセットです。
