# Adobe GenStudio for Performance Marketing Activate

GenStudio for Performance Marketing [!DNL Activate]は、有料広告チャネルを選択するための広告エクスペリエンスのアクティベーションを準備するための一元的な場所を提供します。 _アクティベーション_&#x200B;は、特定の有料チャネルでの配置に合わせて独自にカスタマイズされたクリエイティブ広告エクスペリエンスとして、広告エクスペリエンス、アセット、テキストを準備するプロセスです。 アクティベーションを実行すると、指定されたチャネルに非アクティブな状態でクリエイティブなエクスペリエンスが提供され、チャネルから最終的なレビューと展開の準備が整います。 広告体験を目的の市場に分散できます。

コンテンツエディターとシステムマネージャーは、アセットがGenStudio for Performance Marketingで生成されるか、Adobe Experience Managerにアップロードされるエクスペリエンスを公開できます。

[!DNL Activate]は、サポートされている各ペイドメディアチャネルの製品カードを表示します。 このタイルは有料メディアチャネルを特定し、過去30日間に発生した広告のアクティベーションの成功数をリストします。 また、アクティベーションを管理し、新しいアクティベーションを作成するためのエントリポイントも提供します。

広告エクスペリエンスをアクティベートするには、各ペイドメディアチャネルアカウントをGenStudio for Performance Marketingに接続する必要があります。

## ユースケースをアクティブ化

* 一部の有料広告チャネルに対して、[!DNL Content]から直接承認済みエクスペリエンスをアクティベートします。

* [&#x200B; アクティベーションを作成](create-activation.md)。 有料チャネルにプッシュする前に、ビジュアルアセットを選択し、テキストを割り当て、メタデータを定義することで、広告体験を構築することができます。

* [&#x200B; ライセンス認証の管理](manage-activations.md)。 _[!DNL Activate]_&#x200B;は、アクティブ化されたすべてのエクスペリエンスの概要を提供します。 個々のアクティベーションの詳細を表示することもできます。

## 機能を活用

[!DNL Activate]は、ターゲットの有料チャネルに配信するための広告エクスペリエンスの準備を容易にします。

### コンテンツから直接、承認済みエクスペリエンスをアクティベート

[!DNL Content]から承認済みエクスペリエンスを直接アクティベートすることも、一部の有料チャネルのアクティベーショングループとして複数の承認済みエクスペリエンスをアクティベートすることもできます。 グループ内のあらゆるエクスペリエンスで、同じ有料チャネルをターゲットにする必要があります。

### アクティベーションのためのエクスペリエンスの準備

_エクスペリエンスの設定_ ページでは、広告エクスペリエンスのビジュアル要素を選択し、広告テキスト（call-to-action テキストを含む）を調整し、エクスペリエンスのトラッキングと可視性を向上させる役に立つメタデータを追加するために必要なツールが統合されています。

### 広告体験用のアセットの選択

[!DNL Content]から承認済みビジュアルアセットを検索して選択するか、DropboxまたはOneDriveからアセットをアップロードします。 デバイスから直接アセットをアップロードできます。 1つの広告エクスペリエンスを複数の場所で実行できます。 Assetsは、広告の配置に必要な縦横比でアップロードできます。

### 広告チャネルのエクスペリエンスをプレビューし、アクティベートする

この広告アクティベーションに関連する有料チャネルアカウント、広告セット、およびMeta キャンペーンを確認します。 _プレビュー_ パネルを使用すると、広告エクスペリエンスを確定する前に、様々なプレースメントで表示できます。 （_プレビュー_ パネルでプレースメントを選択すると、広告の表示のみが影響を受けます。 _プレビュー_ パネルでのプレースメントの選択は保存されません）。 広告エクスペリエンスに問題がなければ、**[!UICONTROL 公開]**&#x200B;をクリックして、エクスペリエンスと関連するメタデータを有料広告チャネルにプッシュします。
