# アクティベーションワークフロー

[!DNL Activate]は、MetaやGoogle Campaign Manager 360広告エクスペリエンスなど、チャネル固有のフォーマットでの広告エクスペリエンスのアクティベーションをサポートしています。

GenStudio for Performance Marketingエクスペリエンスとは、広告などのマーケティングキャンペーンコンポーネントのことで、有料広告チャネルやメール上の特定のオーディエンスに対する広告体験として用意されます。 アクティベーション向けのエクスペリエンスは、次の3つの主要な要素で構成されています。

* **メディア アセット**: メディア アセットは、広告エクスペリエンスに含まれる画像（GIF、PNG、JPEG）です。 アクティベーションは現在、静止画像をサポートしています。

  広告体験に適した画像アセットを選択するには、適切なアスペクト比を選択する必要があります。 縦横比は、画像の幅と高さの比例関係を定義し、広告プレースメントの効果に非常に重要です。 ペイドメディアチャネルは、プラットフォーム上の各広告の配置について、有効なアスペクト比を慎重に指定します。 アクティベーションに画像アセットを追加する場合は、エクスペリエンスの最終的な広告配置に基づいてアスペクト比を選択する必要があります。 ファイル形式は、JPEG、PNG、GIFに限定されます。

* **テキスト**: テキストは、見出し、本文、call-to-action要素など、広告に含まれるすべての形式のコピーで構成されます。

* **メタデータ**: コンテンツに割り当てることができるユーザー定義属性。 メタデータは、パフォーマンスの分析、フィルタリング、追跡を強化します。 通常、ユーザーには表示されません。

アクティベーションの作成では、指定されたチャネルへの配置とマーケティングキャンペーンのために、これらの各広告コンポーネントを改良します。 GenStudio for Performance Marketingでは、単一のペイドチャネルに対してひとつのエクスペリエンスをアクティベートすることができます。

## ワークフローのフェーズ

それぞれの有料チャネルには独自の配置の要件がありますが、あらゆる広告のアクティベーションには同じ上位レベルのステップが共有されています。 ペイドチャネルでのエクスペリエンスのアクティベーションには、次の3つのコアフェーズがあります。

1. **GenStudio for Performance Marketingを対象チャネルに接続**。 エクスペリエンスをアクティベートする前に、GenStudioのシステムマネージャーがチャネルアカウントを接続する必要があります。

1. **アクティベーション用にエクスペリエンスを準備**。 活用するためのエクスペリエンスを準備するには、次のふたつの方法があります。

   * 事前定義された設定を使用して、[!DNL Content]から直接承認済みエクスペリエンスをアクティベートします。 これにより、単一のチャネルで複数の広告エクスペリエンスをアクティベートする効率的な方法が実現します。 [!DNL Content] ギャラリーからエクスペリエンスを選択すると、広告エクスペリエンスにアセットを編集または追加することはできません。 [!DNL Content]からのアクティベートは、MetaおよびGoogle Campaign Manager 360広告エクスペリエンスで利用できます。

   * [!DNL Content]からビジュアルアセットを選択し、テキスト要素を追加し、縦横比を選択して、広告エクスペリエンスを構成します。 この手法では、より多くのステップを踏む必要がありますが、よりクリエイティブな柔軟性も高めることができます。 準備には、特定の広告プレースメントに適したアスペクト比でメディアアセットを選択し、call-to-actionのエレメントとボディコピーにテキストを割り当てることが含まれます。 アクティベーション後にユーザーがエクスペリエンスを検索するのに役立つ有益なメタデータを追加できます。 各広告チャネルの配置は、配置に含まれるビジュアルアセットの有効な縦横比を指定します。

1. **エクスペリエンスをレビューしてターゲット チャネルに公開**。 エクスペリエンスのセットアップ中に&#x200B;_プレビュー_ パネルを使用して、アクティベーションを確定する前に、広告プレースメントとテキスト要素の選択を評価します。 最終的な公開前レビューは、宛先チャネルの広告管理アプリで行われます。 例えば、GenStudio for Performance MarketingでMeta広告エクスペリエンスをアクティベートした後は、Meta Ads Managerにログインして広告エクスペリエンスを確認し、特定の属性を選択してから公開する必要があります。

ターゲットの有料メディアチャネルで広告体験が公開されると、[!DNL Insights]はそのパフォーマンスデータを追跡および分析できます。

## サポートされているチャネル

各ペイドメディアチャネルには、独自のアクティベーションワークフローがあります。 アクティベーション用の有料チャネルのガイドラインを選択します。

* [Google Campaign Manager 360](activate-cm360-ad.md)
* [LinkedIn](activate-linkedin-ad.md)
* [Meta](activate-meta-ad.md)
* [ChatGPT](../create/create-chatgpt-ad.md#activate-a-chatgpt-ad)
