# スターターテンプレート

_スターターテンプレート_&#x200B;は、GenStudio for Performance Marketingをすばやく使い始め、効果的なメディアの作成方法を学ぶために設計された、事前定義済みのテンプレートオプションです。 [!DNL Create]で、メディアタイプを選択します。 スターターテンプレートが使用可能な場合は、「**[!UICONTROL スターターテンプレート]**」タブに切り替えて、ニーズに合ったテンプレートを参照して選択します。

![ スターターテンプレートタブ ](/help/assets/starter-templates.png " テンプレートタブの選択"){width="350"}

広告に適切なアスペクト比を選択することは、効果を最大化し、さまざまなプラットフォームやプレースメントをまたいでコンテンツが正しく表示されるようにするために重要です。 アスペクト比を選択する際には、次の点を考慮してください。

- **キャンペーン目標**：目標（ブランド認知度、エンゲージメント、コンバージョン）を定義して、それらをサポートする最適なプレースメントと形式を決定します。
- **プラットフォーム要件** – 各ペイドメディアチャネルは、異なるプレースメントに対してアスペクト比を推奨しています。
- **コンテンツの可視性** – 一部の縦横比はクリエイティブの多くを示し、他の縦横比はコンテンツの切り抜きやサイズ変更を示します。

テンプレートを選択する前に、キャンペーンの目標と配置オプションを確認しましょう。 あらゆるテンプレートにおいて、重要なビジュアルやメッセージを安全地帯に保管し、クリエイティブを完成させる前にプラットフォーム固有の要件を確認し、切り抜きを避けましょう。

## 利用可能なスターターテンプレート

次に、利用可能なスターターテンプレート、サポートされるメディアタイプ、アスペクト比、各メディアチャネルの使用ガイダンスをまとめます。

### LinkedIn広告テンプレート

静止画像では、すべての縦横比がサポートされています。 ビデオの場合は、LinkedIn Storiesに9:16を使用し、ビデオが短い縦長のフォーマットに最適化されていることを確認します。

| 縦横比 | オリエンテーション | ガイダンス |
|--------------|--------------|------------------------------------------------------------------------------------------------|
| 1:1 | 正方形 | デバイスやプレースメント間で一貫した外観。 |
| 1.91:1 | 横 | ニュースフィードまたはスポンサードコンテンツ向けの標準。 重要なビジュアルやメッセージを安全地帯に保管する。 |
| 1:1.91 | 縦 | 縦長の書式。 重要なビジュアルやメッセージを安全地帯に保管する。 |
| 2:3, 4:5 | 縦 | ポートレート形式に適しており、モバイルデバイスでクリエイティブを目立たせるのに役立ちます。 |
| 9:16 | 垂直方向 | LinkedIn ストーリーズに最適（動画向け全画面表示）。 |

### Meta広告テンプレート

すべての縦横比は、静止画像とビデオメディアタイプでサポートされています。

| 縦横比 | オリエンテーション | ガイダンス |
|--------------|--------------|------------------------------------------------------------------------------------------------|
| 1:1 | 正方形 | フィード投稿、カルーセル広告、幅広い互換性を持つフィード動画に適しています。 |
| 1.91:1 | 横 | バナー、リンク広告、ワイドスクリーン動画広告に最適です。 |
| 4:5 | 縦 | モバイルフィードに最適。縦方向のスペースと可視性を最大化します。特にビデオコンテンツに最適です。 |
| 9:16 | 垂直方向 | ストーリーやリール用に設計されており、没入感のあるフルスクリーン体験を提供します。 |
